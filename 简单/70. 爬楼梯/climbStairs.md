### 假设你正在爬楼梯。需要 n 阶你才能到达楼顶。每次你可以爬 1 或 2 个台阶。你有多少种不同的方法可以爬到楼顶呢？
注意：给定 n 是一个正整数

示例1：

```
输入：1
输出：2
解释：有一种方法可以爬到楼顶
1、1阶
```

示例2：

```
输入： 2
输出： 2
解释： 有两种方法可以爬到楼顶。
1.  1 阶 + 1 阶
2.  2 阶
```

示例3：

```
输入： 3
输出： 3
解释： 有三种方法可以爬到楼顶。
1.  1 阶 + 1 阶 + 1 阶
2.  1 阶 + 2 阶
3.  2 阶 + 1 阶
```

LeetCode题目：[70. 爬楼梯](https://leetcode-cn.com/problems/climbing-stairs/)

解法1：数学算法

根据示例，我们可以看出结果为 1 2 3 5 8....，根据规律，从这里我们可以看出，这是一个明显的`斐波那契数列`，那么这样，我们明显可以使用`斐波那契数列`的公式来进行解答

> 公式： ![formula](https://dss1.baidu.com/6ONXsjip0QIZ8tyhnq/it/u=2524561525,4505879&fm=58)

```js
var climbStairs1 = function(n) {
  const sqrt = Math.sqrt(5);
  const fib_n = Math.pow((1 + sqrt) / 2, n + 1) - Math.pow((1 - sqrt) / 2,n + 1);
  return Math.round(fib_n / sqrt);
};
```

解法2：动态规划

从解法1来看，明显的发现了第三项是第二项和第一项的和

- 第i阶可以由一下两中方法得到
  - 在第（i - 1）阶后向上爬一阶
  - 在第（i - 2）阶后向上爬二阶
- 在第i阶的方法总数就是到第（i - 1）和第（i - 2）阶的方法的总和
- 根据题目来说，n是个正整数，所有不用考虑0
  - 官方例子给出了1和2的可能性
    - 直接初始化 `dp[1] = 1`和`dp[2] = 2`，从3开始循环
- 核心公式：dp[i] = dp[i - 1] + dp[i - 2]

```js
let climbStairs = function (n) {
  let dp = []
  dp[1] = 1
  dp[2] = 2

  for (let i = 3; i <= n; i++) {
    dp[i] = dp[i - 1] + dp[i - 2]
  }
  return dp[n]
};
```
