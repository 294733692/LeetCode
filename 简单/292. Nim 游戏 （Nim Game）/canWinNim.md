### 你和你的朋友，两个人一起玩 Nim 游戏：桌子上有一堆石头，每次你们轮流拿掉 1 - 3 块石头。 拿掉最后一块石头的人就是获胜者。你作为先手。

你们是聪明人，每一步都是最优解。 编写一个函数，来判断你是否可以在给定石头数量的情况下赢得游戏

示例:

```
输入: 4
输出: false 
解释: 如果堆中有 4 块石头，那么你永远不会赢得比赛；
     因为无论你拿走 1 块、2 块 还是 3 块石头，最后一块石头总是会被你的朋友拿走。
```

#### LeetCode题目的地址[292. Nim 游戏](https://leetcode-cn.com/problems/nim-game/)

看过博弈论的同学应该都知道，这个题目就是`巴什博弈`，

从简单的分析：（注意，这里的m是最大拿几个）

- 当有`1 - m`个是时，毫无疑问的，先手的必赢。（先手可以全部拿完）
- 当有`m + 1`个时，先手不论拿几个，后手都可以把剩下的石头全部拿完。
- 当有`m + 2 - 2m`时候，先手可以拿走几个，只要剩下`m + 1`个，先手必赢

从这里我们可以发现，只要是面临`m + 1`石头的人一定会失败。

这里可以从小往大推。

- 假设：当前石头数为`n = k * (m + 1) + r`

先手会首先拿走`r`个，接下来假设后手拿走`y`个，先手会拿`m  + 1 - y`个，这样下去，博弈下去，后手的一定会失败

- 假设当前石头数为`n = k * (m + 1)`

假设先手拿`x`个，后手一定会拿`m + 1 - x`个，这样下去先手一定会失败



回到这道题上，这道题最关键的就是，只要给对手留下的数量是4的倍数，就是必赢的.

```js
/**
 * @param {number} n
 * @return {boolean}
 */
let canWinNim = function (n) {
  return (n & 3) !== 0
};
console.log(canWinNim(4));
```
