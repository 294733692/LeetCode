### 罗马数字包含以下七种字符: `I`， `V`， `X`， `L`，`C`，`D` 和 `M`。

`字符          数值
I             1
V             5
X             10
L             50
C             100
D             500
M             1000`

例如， 罗马数字 2 写做 II ，即为两个并列的 1。12 写做 XII ，即为 X + II 。 27 写做  XXVII, 即为 XX + V + II 。

通常情况下，罗马数字中小的数字在大的数字的右边。但也存在特例，例如 4 不写做 IIII，而是 IV。数字 1 在数字 5 的左边，所表示的数等于大数 5 减小数 1 得到的数值 4 。同样地，数字 9 表示为 IX。这个特殊的规则只适用于以下六种情况：

- `I` 可以放在 `V` (5) 和 `X` (10) 的左边，来表示 4 和 9。
- `X` 可以放在 `L` (50) 和 `C` (100) 的左边，来表示 40 和 90。
- `C` 可以放在 `D` (500) 和 `M` (1000) 的左边，来表示 400 和 900。

给定一个罗马数字，将其转换成整数。输入确保在 1 到 3999 的范围内。



通过上面的解释可以得知，例如：`IV`,如果前一位小于后一位，这使用后一位减去前一位的值，`V`大于`I`,这`V-I=4`

思路：

- 枚举出罗马数字对应的阿拉伯数字
- 处理特殊情况：传入值位空或者传入一位罗马数字
- 循环遍历，前一位和后一位做比较，前一位小于后一位，则做减法，<b>循环位数+1</b>，否则返回自身
- 返回输出结果



实现：

```js
/**
 * @param {string} s
 * @return {number}
 */
var romanToInt = function (s) {
  // 枚举出罗马数字对应的数字
  let obj = {
    'I': 1,
    'V': 5,
    'X': 10,
    'L': 50,
    'C': 100,
    'D': 500,
    'M': 1000,
  }
  // 处理特殊情况
  if (s.length === 0) return 0
  if (s.length === 1) return obj[s]

  let result = 0
  for (let i = 0, len = s.length; i < len; i++) {
    let left = s[i]
    let right = s[i + 1]
    // 前一位小于后一位，算出结果赋值给result
    // 循环位数+1,不加一的话会造成最后一位累加
    if (obj[left] < obj[right]) {
      result += obj[right] - obj[left]
      i++
    } else {
      // 结果加前一位
      result += obj[left]
    }
  }
  return result
};
```
